# Code for Health Frontend

这是一个基于 Vite + React + TypeScript 的前端项目。

## 本地运行

前置条件：Node.js 18+

1. 安装依赖：`npm install`
2. 启动开发环境：`npm run dev`
3. 执行生产构建：`npm run build`

## 当前项目结构

- `src/pages`：页面级入口
- `src/layouts`：共享布局
- `src/components`：可复用区块与 UI 组件
- `src/hooks`：主题和路由等逻辑
- `src/lib`：路由常量
- `src/styles`：全局样式
